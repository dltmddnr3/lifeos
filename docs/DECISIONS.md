# DECISIONS

주요 의사결정 기록. 형식: 날짜 - 결정 사항 (이유).

- 2026-07-20: VS Code는 필수로 사용하지 않는다.
- 2026-07-20: GitHub은 백업과 복구 목적으로 사용한다.
- 2026-07-20: 초기 GitHub push는 GitHub Desktop을 사용한다. (Claude Cowork 샌드박스에서 github.com 네트워크 접근이 차단되어 있어 직접 push가 불가능하기 때문)
- 2026-07-20: Slack 채널은 우선 `#lifeos` 하나만 사용한다.
- 2026-07-20: 모든 작업은 Slack 스레드 단위로 관리한다.
- 2026-07-20: 파괴적 작업(GitHub push, 배포, 파일 대량 삭제, DB 파괴적 변경 등)은 사용자 승인이 필요하다.
- 2026-07-20: 기술 스택(LifeOS Manager 구현 방식 포함)은 설계 문서 확정 후 결정한다. 확정 전까지는 TBD로 표시한다.
