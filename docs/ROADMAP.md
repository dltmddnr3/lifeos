# ROADMAP

각 Phase의 기술 스택과 세부 범위는 아직 확정되지 않았다(TBD). 설계가 확정되는 대로 이 문서를 갱신한다.

## Phase 0 — 협업 기반 (현재)

사용자·ChatGPT CTO Agent·Claude Engineer·LifeOS Manager가 Slack에서 협업하는 파이프라인의 기준 문서(PROJECT.md, ARCHITECTURE.md, WORKFLOW.md, DECISIONS.md, CLAUDE.md)를 정의한다. 코드 구현은 아직 시작하지 않는다.

## Phase 1 — LifeOS Core

협업 파이프라인(Slack ↔ LifeOS Manager ↔ ChatGPT CTO Agent ↔ Claude Engineer ↔ GitHub)을 실제로 동작하는 최소 버전으로 구현한다.

## Phase 2 — Second Brain

정보를 저장·검색·연결하는 개인 지식 관리 기능을 구축한다.

## Phase 3 — Task & Project OS

작업(Task)과 프로젝트를 관리하는 기능을 구축한다.

## Phase 4 — Calendar & Reminder

일정 관리와 리마인더 기능을 통합한다.

## Phase 5 — Specialized Agents

목적별 전문 에이전트(예: 리서치, 재무, 건강 등)를 추가한다.

## Phase 6 — 완전 모바일 운영

아이폰만으로 LifeOS 전체를 운영할 수 있는 상태를 완성한다.
