# PROJECT

## 목표

아이폰만으로 모든 개발과 운영이 가능한 AI 시스템(LifeOS) 구축.

## 역할 분담

| 역할 | 담당 | 책임 |
|---|---|---|
| CTO | ChatGPT | 프로젝트 설계, 아키텍처 결정 |
| Lead Software Engineer | Claude | 설계 구현, 코드 작성/수정, 테스트, Git 커밋/푸시, 프로젝트 구조 관리, 오류 수정, 리팩토링 |
| Owner | 사용자 | 방향 설정, 설계안 전달, 최종 승인 |

## 작업 원칙

1. 최대한 GUI 기반으로 작업한다.
2. 사용자의 터미널 직접 사용을 최소화한다.
3. 필요한 명령은 Claude가 직접 실행한다.
4. 프로젝트 구조는 항상 유지보수하기 쉽게 관리한다.
5. 변경 전에는 항상 변경 계획을 설명한다.
6. 큰 변경은 작은 단위로 나누어 진행한다.
7. Git 커밋은 의미 있는 단위로 수행한다.
8. README와 프로젝트 문서는 항상 최신 상태를 유지한다.

## 환경 노트

- 개발 위치: Mac mini, 경로 `~/developer/lifeos`
- Claude의 명령 실행 환경은 격리된 Linux 샌드박스이며, 연결된 프로젝트 폴더만 마운트되어 파일이 실제 Mac에 반영된다. Xcode 등 macOS 전용 도구는 이 샌드박스에서 사용할 수 없다.
- 샌드박스의 외부 네트워크 접근이 기본적으로 제한되어 있어, GitHub 인증/clone/push는 별도 조치(네트워크 허용 설정 변경 또는 대체 방법) 후 진행한다.

## Phase 로드맵

- **Phase 0 (현재)**: 개발 환경 구축 — 프로젝트 골격, 문서, Git 저장소 연결
- **Phase 1**: 기술 스택 확정 (ChatGPT 설계안 기반)
- **Phase 2**: 이후 설계에 따라 갱신
